## Create a catalix project

Run the `init` command inside a **Next.js** project to initialize a Catalix project.

```package-install
npx @catalix/cli@latest init
```

## Add Components

Run the `init` command inside a **Next.js** project to initialize a Catalix project.

```package-install
npx @catalix/cli@latest add button
```
