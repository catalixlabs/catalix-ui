## Installation

```package-install
npx @catalix/cli@latest add button
```

## Usage

```tsx title="page.tsx" lineNumbers
import Button from "@/components/core/button"; //[!code ++]

export default function Example() {
  return <Button>Click me</Button>; //[!code ++]
}
```

## API Reference

| Prop     | Type                                                                                    | Default   | Description                            |
| -------- | --------------------------------------------------------------------------------------- | --------- | -------------------------------------- |
| variant  | `"default"` \| `"destructive"` \| `"outline"` \| `"secondary"` \| `"ghost"` \| `"link"` | `default` | Defines the visual style of the button |
| size     | `"sm"` \| `"md"` \| `"lg"`                                                              | `md`      | Controls the button size               |
| ...props | `ButtonPrimitiveProps`                                                                  | \_        | All standard button props              |
